What is Git?

Git é um sistema de controle de versão open-source e gratuito. Por meio dele, é possível observar a criação de histórico com versões das alterações realizadas no código-fonte dos projetos de desenvolvimento. Inclusive, foi criado por Linus Torvalds, que também é conhecido como o criador do Linux.

Advantages of using git:

  • Be able to get older versions of your projects if you have messed up on them!

  • Don't loose your files, save everything!

  • Sync with Github and get to see your work all set!

Git Commands

Iniciar um Repositório

git init

Com este comando você podera criar o seu primeiro repositórtio GIT

Mudar a sua Branch

git branch -m main

Por padrão o git lhe coloca na branch master mas o ideal por boas praticas é ultilizar a branch main

Adicionar novos arquivos 

git add (nome do arquivo)

Este comando serve para que o git possa rastrear o arquivo que você acabou de alterar

Fazer um commit

git commit

Um terminal ira aparecer e você precisa digitar a mensagem que ficará registrada nesse commit

Mandar para o Repositório Remoto

git push --set-upstream main

Após rodar esse comando, seus arquivos irão para seu repositórtio remoto EX.: Github, GitLab

Mostrar Historico de Commits

git log

Mostrará todos os commits realizados antes do push e com seus HASH, você pode navegar pelo historico assim
